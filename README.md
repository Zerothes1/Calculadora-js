# Calculadora Web Dinámica (HTML5, CSS3 & JavaScript)

Un proyecto práctico de desarrollo Frontend que implementa una interfaz limpia y un motor de cálculo aritmético funcional ejecutado directamente en el navegador.

Este proyecto demuestra el uso de buenas prácticas en desarrollo web estructurado, separando por completo la estructura (HTML), la capa de estilos de usuario (CSS) y la lógica algorítmica (JavaScript).

---

## Características y Funcionalidades
* **Diseño Moderno y Responsivo:** Estilos limpios y centrados mediante CSS autónomo (`style.css`), optimizando la experiencia de usuario con transiciones suaves en los campos de entrada y efectos interactivos (*hover*) en los botones.
* **Procesamiento Dinámico:** Manipulación directa del DOM (Document Object Model) mediante JavaScript (`index.js`) para capturar las entradas del usuario y renderizar los resultados instantáneamente.
* **Control Aritmético:** Validación y ejecución de operadores estándar utilizando una lógica basada en estructuras de control condicional (`switch`).

---

## Tecnologías Aplicadas
* **HTML5:** Estructuración semántica de componentes y campos de captura de datos.
* **CSS3:** Hoja de estilos externa para la arquitectura visual, tipografía moderna y diseño responsivo.
* **JavaScript (ES6):** Manejo de eventos (`onclick`), conversión de tipos de datos (`parseFloat`) y manejo de lógica computacional.

---

## Estructura del Repositorio
* `index.html` -> Arquitectura y estructura de las cajas de texto y el botón de acción.
* `style.css` -> Reglas de diseño visual, márgenes, colores y transiciones.
* `index.js` -> Captura de variables del DOM y procesamiento del motor matemático.

---

## Instrucciones de Ejecución Local
1. Descarga o clona este repositorio en tu equipo.
2. Navega hasta la carpeta del proyecto.
3. Abre el archivo `index.html` con un doble clic en cualquier navegador web moderno (Google Chrome, Microsoft Edge, Brave, Mozilla Firefox).
4. Ingresa los valores numéricos, especifica el operador matemático (`+`, `-`, `*`, `/`) y presiona el botón **Calcular**.
